# Life OS — AI駆動の業務管理システム

## 概要

AIエージェント（Claude Code）との対話だけで構築した、個人の業務管理システム。
コーディング経験なし・非エンジニアの立場から、以下を実現。

## 全体アーキテクチャ

```
┌─────────────┐
│  NOTES.md   │ ← 何でもメモする入力口
└──────┬──────┘
       │ AIが自動分類（FileChanged hook）
       ▼
┌──────────────────────────────────────────┐
│                                          │
│  ┌───────────┐  ┌─────────┐  ┌────────┐ │
│  │ IDEAS.md  │  │ Linear  │  │ memory │ │
│  │ アイデア   │  │ タスク   │  │ 記憶   │ │
│  │ 戦略蓄積  │  │ 実行管理 │  │ 永続化 │ │
│  └───────────┘  └─────────┘  └────────┘ │
│                                          │
└──────────────────────────────────────────┘
       │
       ▼ 更新時に自動通知
┌──────────────────────────┐
│  Slack    │  Notion      │
│  通知     │  ドキュメント │
└──────────────────────────┘
       │
       ▼ 常時
┌──────────────┐
│   GitHub     │ ← auto-push hookで自動バックアップ
└──────────────┘
```

## 情報の流れ

### 1. 入力（NOTES.md）
ユーザーがメモを書く。内容は何でもOK（タスク、アイデア、事実、質問）。

### 2. 自動分類（AI + hook）
FileChanged hookがNOTES.mdの変更を検知し、AIエージェントが内容を判断して振り分け：
- タスク → LinearにIssue作成
- アイデア・戦略 → IDEAS.mdの該当セクションに追記
- 覚えてほしい事実 → memoryに保存
- 質問 → その場で回答

### 3. アイデア評価（IDEAS.md）
Impact（影響度）× Ease（実行しやすさ）のマトリクスで評価。

```
| アイデア | Impact | Ease | メモ |
|---|---|---|---|
| バリュエーション最新化 | A | A | CCで即実行 |
| 投資基準の体系化 | A | C | 設計が必要 |
```

ユーザーがAA/AB/BAのものを選び、CCがLinearにタスク化。

### 4. タスク実行（Linear）
AIエージェントがLinear MCPを通じて直接操作：
- Issue作成・更新・ステータス変更
- ラベル・優先度の自動設定
- プロジェクト単位での中長期管理

### 5. 自動通知（Slack）
更新があるとSlack #cc-tipsに自動投稿。チーム共有に活用。

### 6. ドキュメント共有（Notion）
ナレッジをNotionに蓄積。新着テーブルで確認状況を管理。

### 7. 自動バックアップ（GitHub）
PostToolUse hookでgit commit時に自動push。データ消失リスクをゼロに。

## 使用技術・ツール

| 技術 | 用途 |
|---|---|
| Claude Code | AIエージェント（全体の設計・実装・運用） |
| Claude Code hooks | ファイル変更検知、自動push |
| MCP（Model Context Protocol） | Linear, Slack, Notion, Google Calendar連携 |
| Linear | タスク管理 |
| Slack | 通知・チーム共有 |
| Notion | ドキュメント管理 |
| GitHub | バージョン管理・バックアップ |
| Google Calendar | スケジュール管理 |

## 成果

- **非エンジニアが1日で構築**: コーディング経験ゼロから、AIとの対話だけで業務システムを構築
- **5つのSaaSツールを統合**: Linear, Slack, Notion, GitHub, Google Calendarを一元管理
- **自動化**: メモの分類、バックアップ、通知が全て自動化
- **再現可能**: 同じ手法で他の企業・個人にも導入可能
